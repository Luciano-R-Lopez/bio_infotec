Sensores del Bio-InfoMonitor

El sistema Bio-InfoMonitor utiliza múltiples sensores integrados para medir los parámetros clave del biodigestor. Entre ellos:

  • Sensor DHT11: Mide la humedad y temperatura ambiente.
  • Sensor DS18B20: Mide la temperatura del líquido dentro del biodigestor.
  • Sensor de Caudal YF-S401: Mide el flujo del biogás producido.

Calendario de Eventos

El calendario permite agendar eventos relacionados con el mantenimiento y monitoreo del biodigestor. Puedes seleccionar fechas importantes como:

  • Mantenimiento de sensores.
  • Verificación de producción de biogás.
  • Control de calidad del biogás.

Asegúrate de programar revisiones regulares para optimizar la producción de biogás.
